# Board Presentation Template

> Instructions: This is a slide-by-slide structure for Board Presentations. 10-15 slides, one core message per slide. Use the SCQA framework to open, and organize content using the Pyramid Principle.

---

# Board Presentation Structure Guide

**Total Duration**: 20-30 minutes (Presentation 15-20min + Q&A 10-15min)
**Total Slides**: 10-15 content slides + Appendix
**Core Principle**: Summarize the core message of each slide in one sentence (Page Title = Key Message)


## Slide 1: Title Slide
- Project Name
- Presenter / Team
- Date
- Classification


## Slide 2: Executive Summary — The Most Important Slide

**Core Message**: One-sentence summary of the entire presentation

Page Content:
- Left Column (40%): 3-5 bullet points
  - Current State / Problem: 1-2 sentences
  - Recommendation: 1 sentence
  - Expected Return: 1 sentence + key numbers
  - Decision Request: 1 sentence
- Right Column (60%): 1 key chart
  - Recommended: Waterfall chart (TCO comparison) / Bar chart (ROI) / Roadmap

3-Second Test: The reader should grasp the core message in a 3-second glance
30-Second Test: The entire story should be tellable in 30 seconds


## Slide 3-4: Why Change — Burning Platform

**Slide 3: External Pressures**
Core Message: "Our industry is undergoing fundamental change"
- 1 industry trend chart (3-4 trends)
- "What this means for us" (1-2 sentences per trend)

**Slide 4: Internal Current State**
Core Message: "If we don't change, we face a [$X]M cost"
- 3-5 key data points showing the gap
  - Us vs. Industry Average vs. Industry Leader
- "5-Year Cumulative Cost of Inaction" — 1 number


## Slide 5-6: What We Recommend

**Slide 5: Strategic Blueprint**
Core Message: "Recommend transforming from [Current State] to [Target State]"
- 1 target architecture / capability blueprint (not too technical)
- 3-5 core initiatives, each with 1-2 lines of description

**Slide 6: Core Initiative Details**
Core Message: "Three core initiatives will drive [X]% of expected returns"
- Initiative 1: [Name] + Investment $[X]M + Expected Return $[X]M/year + Timeline
- Initiative 2: [Same format]
- Initiative 3: [Same format]


## Slide 7-8: Investment & Returns

**Slide 7: Investment Overview**
Core Message: "Total investment $[X]M, over [X] years, Year 1 requires $[X]M"
- Investment waterfall chart (by category and year)
- Comparison with industry benchmarks ("Our IT investment ratio will shift from X% to Y%")

**Slide 8: Expected Returns**
Core Message: "5-Year net benefit $[X]M, payback period [X] years"
- NPV / IRR / Payback — three key numbers prominently displayed
- 5-Year cumulative cash flow chart
- "Do Nothing" vs. "Invest" comparison line


## Slide 9-10: How to Execute

**Slide 9: 3-Year Roadmap**
Core Message: "Roll out in three waves, each focusing on [X] core initiatives"
- Timeline: Wave 1 (Quick Wins 0-6m) → Wave 2 (Core 6-18m) → Wave 3 (Innovation 18-36m)
- Per Wave: Key initiatives + Milestones + Investment + Expected value

**Slide 10: Key Milestones & Dependencies**
Core Message: "5 key milestones in the first 6 months"
- 5-7 milestone timeline (including decision points)
- Key dependencies (e.g., "Board approval of Year 1 budget required to launch")


## Slide 11: What We Need From You (Decision Request)

Core Message: "Requesting Board approval on the following three decisions"
1. [Decision 1: e.g., Approve IT strategic direction and 3-year investment framework]
2. [Decision 2: e.g., Approve Year 1 budget of $[X]M]
3. [Decision 3: e.g., Authorize CEO/CIO to make autonomous decisions within [X] scope]

If decisions are time-sensitive, note: "Recommend deciding by [date] to catch the [critical time window]"


## Slide 12: Q&A


## Slide 13+: Backup Appendix (Do not present proactively; flip to only if asked)
- Detailed financial model
- Industry benchmark detailed data
- Key risks & mitigation matrix
- Competitive intelligence
- Team introduction
- Glossary

## Board Presentation Skills Checklist

### Before the Meeting
- [ ] Understand each person's focus (CEO cares about strategy / CFO cares about money / Independent directors care about risk / ...)
- [ ] Conduct one-on-one pre-meetings with key directors to get informal feedback and address individual concerns
- [ ] Conduct at least one internal dry run
- [ ] Prepare answers for the 3 most likely tough questions
- [ ] Verify all numbers: source, calculation method, cross-validation for every figure

### During the Presentation
- [ ] First 30 seconds: Give the conclusion directly — "We recommend investing $X, with an expected 5-year return of $Y"
- [ ] Don't read from the slides (PPT is for the audience, not your script)
- [ ] Spend only 1-2 minutes per slide, don't stay on any slide for more than 3 minutes
- [ ] When asked something you don't know: "That's an important question. I'll confirm and provide a written response today" — better than bluffing
- [ ] Watch board members' facial expressions and body language — if someone is frowning, proactively ask "Do you have any concerns?"

### After the Meeting
- [ ] Send meeting minutes and supplementary materials within 24 hours
- [ ] Clarify follow-up actions and timing for each decision request
- [ ] If a decision is deferred, understand the reasons and provide targeted supplementary information

### Most Common Questions (Prepare Answers in Advance)
1. "What if we do nothing?" → Have "Cost of Inaction" data ready
2. "Why $X, and not less?" → Have the scope vs. cost relationship ready
3. "How are you sure we can save/earn this much?" → Have sensitivity analysis ready
4. "What are competitors doing?" → Have competitive benchmarking ready
5. "What's the biggest risk? What if it fails?" → Honestly present risks + mitigation + stop-loss line
6. "Why now? Can we wait six months?" → Have timing urgency argument ready
