# Change Readiness Assessment Tool

> Instructions: Use this tool at project initiation and before key milestones to assess the organization's readiness for upcoming change. Scores = early signals for discovering risks, not judging "can we do it" but rather "where do we need to invest more effort."

## Score Interpretation

| Score | Rating | Interpretation | Recommendation |
|------|------|------|------|
| 4.5 - 5.0 | ✅ Green | Organization is ready | Proceed as planned, maintain current leadership support and communication cadence |
| 3.5 - 4.4 | 🟡 Yellow | Basically ready, but has weak areas | Develop reinforcement plan targeting the lowest-scoring dimensions, reassess every 2 weeks |
| 2.5 - 3.4 | 🟠 Orange | Insufficient readiness, higher risk | Postpone major change actions, invest 2-3 months first to address readiness issues |
| 1.0 - 2.4 | 🔴 Red | Organization is not yet ready | Forcing ahead = high probability of failure; need to start by building urgency and leadership alignment |

## Usage Scenarios

- **Before project launch**: Assess the starting point of change, decide whether to invest time up front to improve readiness
- **Key decision points**: e.g., whether to proceed to the next phase, supported by readiness data
- **When facing resistance**: Reassess to see which dimension's readiness has declined
- **Project retrospective**: Compare initial and final readiness, summarize "what kind of change succeeds best in this organization"

---

## Key Reminders

1. This is not a score-it-and-set-it-aside exercise — Low-scoring dimensions must have specific improvement actions
2. Be honest when assessing — self-deception = planting the seeds of project failure
3. Multi-role assessment — have Sponsor/PM/Business Rep/Frontline Rep score separately, see where disagreements lie
4. Trend matters more than absolute score — rising from 2.5 to 3.5 is more valuable than staying at 4.0
5. This tool assesses "readiness", not "whether it's worth doing" — a score of 2 doesn't mean don't do it, it means invest in readiness first
